### Backend ([`task2_13_may_2026/server`](task2_13_may_2026/server))

- **Dependencies**: `zod`, `multer`, `mammoth`, `cors`
- **Config**: [`src/constants/llm.js`](task2_13_may_2026/server/src/constants/llm.js) — `OPENAI_MODEL` (default `gpt-4.1`); [`.env.example`](task2_13_may_2026/server/.env.example) documents `OPENAI_MODEL`
- **Model**: [`src/models/Project.js`](task2_13_may_2026/server/src/models/Project.js) — `title`, `requirementsRaw`, optional `sourceFile`, `status` (`pending_clarification` | `ready` | `failed`), `clarificationQuestions`, `clarificationAnswers`, `tickets`, `lastError`
- **Zod**: [`src/schemas/llmSchemas.js`](task2_13_may_2026/server/src/schemas/llmSchemas.js) — LLM shapes + answers body (including **Other** requires `otherText`)
- **LLM**: [`src/services/clarificationLlm.js`](task2_13_may_2026/server/src/services/clarificationLlm.js) and [`src/services/ticketsLlm.js`](task2_13_may_2026/server/src/services/ticketsLlm.js) — `getOpenAIClient()`, `gpt-4.1`, `response_format: json_object`, up to **2** attempts on parse/validation failure; server assigns `questionId` (UUID) and ticket keys `TKT-1`, …; dependencies resolved from **titles** to **keys**
- **Upload / text**: [`src/services/extractRequirementsText.js`](task2_13_may_2026/server/src/services/extractRequirementsText.js) — `.txt`, `.md`, `.docx` only; merge paste + file with a separator
- **CSV**: [`src/services/ticketsCsv.js`](task2_13_may_2026/server/src/services/ticketsCsv.js)
- **API**: [`src/routes/projectRoutes.js`](task2_13_may_2026/server/src/routes/projectRoutes.js) + [`src/routes/api.js`](task2_13_may_2026/server/src/routes/api.js)  
  - `POST /api/projects` (multipart: `title`, `requirements`, optional `file`)  
  - `GET /api/projects`, `GET /api/projects/:id`  
  - `POST /api/projects/:id/clarifications` — saves answers, then generates tickets; on LLM failure → `failed` + `lastError`, answers kept for retry  
  - `GET /api/projects/:id/export.csv` — when `ready`
- **App**: [`src/app.js`](task2_13_may_2026/server/src/app.js) — CORS + `app.use("/api", apiRoutes)`
